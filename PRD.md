# PRD: Trend Radar (트렌드 레이더)

## 1. 개요

### 1.1 제품 비전
**"유튜브 쇼츠 자동화를 위한 트렌드 & 바이럴 콘텐츠 탐지기"**

유튜브 쇼츠 자동화 프로그램 개발 시 참고할 수 있는 트렌드 키워드와 바이럴 영상을 탐지하는 대시보드 서비스

### 1.2 목표 사용자
| 페르소나 | 설명 | 핵심 니즈 |
|----------|------|-----------|
| **Primary** | 유튜브 쇼츠 자동화 개발자 (본인) | 떡상 콘텐츠 패턴 파악, 트렌드 키워드 수집 |
| Secondary | 쇼츠 크리에이터 | 바이럴 콘텐츠 벤치마킹 |

### 1.3 핵심 가치 제안 (Value Proposition)
1. **바이럴 탐지** - 구독자 적은데 조회수 폭발한 영상 발견
2. **트렌드 파악** - 지금 뜨는 키워드 실시간 확인
3. **패턴 분석** - 어떤 콘텐츠가 터지는지 파악

### 2.1 MVP (Phase 1) - 현재 개발 범위

#### F1. Google Trends 트렌드 키워드
| 항목 | 설명 |
|------|------|
| 기능 | 실시간 급상승 검색어 표시 |

| 표시 | 키워드 목록, 검색량 변화 |
| 용도 | 쇼츠 주제/해시태그 선정 참고 |

#### F2. 유튜브 바이럴 쇼츠 탐지 ⭐ 핵심
| 항목 | 설명 |
|------|------|
| 기능 | 구독자 대비 조회수 폭발 영상 탐지 |
| 탐지 기준 | 구독자 1만 이하 + 조회수 10만 이상 (조정 가능) |
| 지표 | 조회수/구독자 비율, 업로드 후 경과 시간 |
| 표시 | 썸네일, 제목, 채널, 구독자수, 조회수, 비율 |

**바이럴 비율 계산:**
```
바이럴 비율 = 조회수 / 구독자수
예) 구독자 5,000 / 조회수 500,000 = 100x (100배 바이럴)
```

### 4.2 YouTube 바이럴 탐지 로직

```typescript
interface ViralVideo {
  id: string;
  title: string;
  channelTitle: string;
  subscriberCount: number;   // 구독자 수
  viewCount: number;         // 조회수
  viralRatio: number;        // 조회수/구독자 비율
  publishedAt: string;       // 업로드 시간
  thumbnailUrl: string;
}

// 바이럴 판정 기준 (조정 가능)
const VIRAL_CRITERIA = {
  maxSubscribers: 10000,     // 구독자 1만 이하
  minViews: 100000,          // 조회수 10만 이상
  minViralRatio: 10,         // 최소 10배 이상
};

function isViralVideo(video: ViralVideo): boolean {
  return (
    video.subscriberCount <= VIRAL_CRITERIA.maxSubscribers &&
    video.viewCount >= VIRAL_CRITERIA.minViews &&
    video.viralRatio >= VIRAL_CRITERIA.minViralRatio
  );
}
```

### 4.3 API Rate Limits

| YouTube Data API | 10,000 units/day | 30분 캐싱 |
| Google Trends | 비공식 | 1시간 캐싱 |
